# 使用可能なポートを探す

## 現在の状況

多くのポートが使用中です：
- ポート5000: AirPlay Receiver
- ポート5001: 以前のPythonプロセス
- ポート8080: 別のPythonアプリケーション
- ポート3000: 使用中

## 使用可能なポート

以下のポートが使用可能です：
- **ポート4000**: 使用可能 ✅
- **ポート5002**: 使用可能 ✅（推奨）
- **ポート8000**: 使用可能 ✅
- **ポート9000**: 使用可能 ✅

## 設定方法

`.env` ファイルに以下を設定：

```bash
PORT=5002
```

## 起動方法

```bash
source venv/bin/activate
python3 app.py
```

## ポートを確認するコマンド

```bash
# 特定のポートが使用中か確認
lsof -i :5002

# 複数のポートを確認
for port in 4000 5002 8000 9000; do
  lsof -i :$port >/dev/null 2>&1 && echo "ポート$port: 使用中" || echo "ポート$port: 使用可能"
done
```

## 推奨ポート

**ポート5002** を推奨します（`.env` に設定済み）。
