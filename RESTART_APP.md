# アプリケーション再起動ガイド

修正を反映するために、アプリケーションを再起動してください。

## 手順

### 1. 現在実行中のアプリを停止

ターミナルで `Ctrl+C` を押してアプリを停止します。

### 2. アプリを再起動

```bash
# 仮想環境を有効化（まだ有効化していない場合）
source venv/bin/activate

# アプリを起動
python3 app.py
```

### 3. 動作確認

別のターミナルで以下を実行：

```bash
curl http://localhost:5002/health
```

`{"status":"ok",...}` が返ってくれば正常です。

## 修正内容

- FXクエリ（"USD/JPY"、"usdjpy"など）は内部のFX分析AIエージェントで処理されます
- プレースホルダーURL（`example.com`など）の外部AIは呼び出されません
- データが見つからない場合は、適切なエラーメッセージが表示されます
