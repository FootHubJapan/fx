# ポート5002使用中の問題解決

## 問題

ポート5002が既に使用中で、アプリが起動できません。

## 解決方法

### 方法1: プロセスを手動で停止（推奨）

ターミナルで以下を実行：

```bash
# ポート5002を使用しているプロセスを確認
lsof -i :5002

# プロセスID（PID）を確認して、停止
kill -9 <PID>

# 例: PIDが44615の場合
kill -9 44615
```

### 方法2: 別のポートを使用

`.env`ファイルを編集して別のポートを使用：

```bash
# .env ファイルを編集
PORT=5003
```

その後、アプリを起動：

```bash
python3 app.py
```

### 方法3: すべてのPythonプロセスを確認

```bash
# 実行中のPythonプロセスを確認
ps aux | grep python

# 必要に応じて停止
kill -9 <PID>
```

## 確認

ポートが解放されたか確認：

```bash
lsof -i :5002
```

何も表示されなければ、ポートは使用可能です。
